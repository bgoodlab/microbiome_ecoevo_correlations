Analysis and figure generation code for the manuscript, B.H. Good, L.B. Rosenfeld. Eco-evolutionary feedbacks in the human gut microbiota. biorXiv 2022.01.26.477953.

All analysis can be redone and figures regenerated by running the command "python generate_all_figures.py" from the base directory. 

Expected runtime: <10 minutes

Code tested with:

Python 3.9.12
numpy 1.21.5
scipy 1.7.3
matplotlib 3.5.2
BioPython 1.78
statsmodels 0.13.2
